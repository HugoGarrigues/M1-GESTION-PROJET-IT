# VisionFood – Projet Gestion de Projet IT

Ce dépôt a été réalisé dans le cadre du module **Gestion de Projet IT**.

Notre équipe était composée de :

- **Thomas Iafrate**
- **Hugo Garrigues**
- **Matt Crayston**

Le projet consiste à cadrer une application nommée VisionFood, visant à aider les utilisateurs à réduire leur gaspillage alimentaire en gérant leurs stocks et en cuisinant les produits disponibles.
